# Contract Addresses

All contract addresses for the Livepeer protocol.

## Addresses

The addresses for the protocol contracts are provided in a simple import with types.

<SyncedTabs group="lang" items={['React', 'React Native']}>
  <SyncedTab>
    ```tsx filename="React"
    import {
      arbitrumOneAddress,
      arbitrumRinkebyAddress,
      mainnetAddress,
      rinkebyAddress,
    } from '@livepeer/react';
    ```

  </SyncedTab>
  <SyncedTab>
    ```tsx filename="React Native"
    import {
      arbitrumOneAddress,
      arbitrumRinkebyAddress,
      mainnetAddress,
      rinkebyAddress,
    } from '@livepeer/react-native';
    ```

  </SyncedTab>
</SyncedTabs>

These addresses will be expanded upon in a future version of livepeer.js to include more in-depth contract
interaction.

## Chain IDs

There are exports for the chain IDs the Livepeer protocol is currently deployed on. This includes both
Arbitrum One/mainnet and testnet chains.

<SyncedTabs group="lang" items={['React', 'React Native']}>
  <SyncedTab>
    ```tsx filename="React"
    import {
      allChainId, mainnetChainId, testnetChainId
    } from '@livepeer/react';
    ```

  </SyncedTab>
  <SyncedTab>
    ```tsx filename="React Native"
    import {
      allChainId, mainnetChainId, testnetChainId
    } from '@livepeer/react-native';
    ```

  </SyncedTab>
</SyncedTabs>
